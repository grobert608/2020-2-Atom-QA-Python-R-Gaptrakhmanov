# Bash script

* Принимает на вход имя файла, который необходимо обработать `./parser.sh <file_name>`
* Результаты работы скрипта - в файле `result_parse.txt`
* Если файл не существует, то скрипт завершается с ошибкой.

# Python script

* Запускается через `./parser.py`
* Принимает на вход имя файла по ключу `--file`
* Если необоходимо обработать все файлы из дирректории, то принимает путь по ключу  `--dir`
* Если необоходимо вывести в формате `json`, то передается ключ `--json 1`
* Сохраняет соответвубщие файлы в формате `.txt` и `.json`
